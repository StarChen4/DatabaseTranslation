# 军事数据库翻译原则（更新版）

## 1. 保留型号/编号
保留所有型号和编号，例如：
- "J-16" → "J-16"
- "F-11" → "F-11"
- "SH-60B" → "SH-60B"

## 2. 保留缩写，在缩写后括号中加翻译
例如：
- "AAM" → "AAM(空对空导弹)"
- "LRASM" → "LRASM(远程反舰导弹)"

## 3. 保留项目代号/计划名称，如果有含义的在后加括号中翻译注释，无含义不翻译（比如音译）
例如：
- "MIG-23BN Flogger-H" → "MIG-23BN Flogger-H(鞭笞者-H)"

## 4. 保留制造商名字，除非已有明确的中文名字对应
例如：
- "4x Waukesha Diesels" → "4台Waukesha柴油发动机"

## 5. 保持人名为英文，但要翻译他的军衔，如果是舰艇、潜艇，则在后面加"号"
例如：
- "T-AK 3008 2nd Lt. John P. Bobo" → "T-AK 3008 John P. Bobo少尉号"

## 6. 保留单位制
例如：mm, km, kg, kT, mT等

## 7. 翻译固定译名
例如：
- "Falcon" → "战隼"
- "Sidewinder" → "响尾蛇"
- "Seahawk" → "海鹰"
- "Phalanx" → "密集阵"

## 8. 翻译除上述说明外的英文单词
例如：
- "Single Rails" → "单联挂架"

## 9. Comments里若出现句子或段落则完整翻译

## 10. 当译名翻译不确定时，记录并保留原文

## 11. 注意军事术语的使用和翻译准确性、专业性，禁止文学化

## 12. 方括号内容的处理原则

方括号通常用于表示：
- **替代型号**：保留所有型号信息
  - 示例：`OH-6D Cayuse [MD-500D, Model 369D]` → `OH-6D Cayuse [MD-500D, Model 369D]`
- **级别/类别**：翻译类别，保留型号
  - 示例：`LD 601 Tarlac [Makassar Class]` → `LD 601 塔拉克号 [Makassar级]`
- **技术规格**：翻译功能描述，保留技术型号
  - 示例：`127mm/64 Vulcano GLR IR [Guided Long Range]` → `127mm/64 Vulcano GLR IR [制导远程]`

**翻译原则**：保留方括号结构，根据内容类型分别处理

## 13. 北约代号（NATO Codename）的处理
**发现依据**：18条样本包含北约代号

常见北约代号包括：Flanker（侧卫）、Bear（熊）、Hind（雌鹿）、Fishcan等

**翻译原则**：
- **有固定中文译名的**：保留原文，可在括号中添加中文译名
  - 示例：`Su-30MK2 Flanker G` → `Su-30MK2 Flanker G(侧卫G型)`
  - 示例：`Mi-24A Hind A` → `Mi-24A Hind A(雌鹿A型)`
- **无固定译名或译名不常用、不确定的**：保留原文
  - 示例：`J-7 Fishcan` → `J-7 Fishcan`

---

## 14. 口径/规格斜杠格式（mm/caliber）的处理
**发现依据**：18条样本包含此格式

格式通常为：`口径mm/倍径 型号 类型`

**翻译原则**：
- 保留口径和倍径数字
- 保留型号编号
- 翻译武器类型和功能描述
- 示例：
  - `127mm/64 Vulcano GLR IR [Guided Long Range]` → `127mm/64 Vulcano GLR IR [制导远程]`
  - `155mm/39 M109A4+ Self-Propelled Howitzer` → `155mm/39 M109A4+自行榴弹炮`
---

## 15. 舰船命名规则
**发现依据**：14条样本采用标准舰船命名格式

格式通常为：`舰种代号 编号 舰名 [级别]`

**翻译原则**：
- 保留舰种代号（CVN, DDG, LST等）
- 保留编号
- 翻译舰名，在后加"号"
- 翻译级别为"XX级"
- 示例：
  - `CVN 65 Enterprise` → `CVN 65 企业号`
  - `LST 1179 Newport` → `LST 1179 纽波特号`
  - `LD 601 Tarlac [Makassar Class]` → `LD 601 塔拉克号 [Makassar级]`

---

## 16. 数量表达式的处理
**发现依据**：28条样本的Comments中包含数量表达

常见格式：`数字x`、`数字 pr Bty`、`数字 Cells`、`数字 rnds`

**翻译原则**：
- `数字x` → `数字台/座/门`（根据装备类型）
- `pr Bty` → `每连`
- `Cells` → `单元`
- `rnds` → `发`
- `Mounts` → `座`
- 示例：
  - `4x` → `4座`
  - `2x pr Bty` → `每连2座`
  - `[16 Cells]` → `[16单元]`

## 18. 技术状态标识的处理
**发现依据**：6条样本包含状态标识

常见标识：
- `DEPRECATED` → 已弃用
- `Cancelled` → 已取消
- `Retired` → 已退役
- `Mod` / `Modified` → 改进型
- `Upgraded` → 升级型

**翻译原则**：
- 完整翻译状态标识
- 示例：
  - `Comments: DEPRECATED` → `注释：已弃用`
  - `Comments: Cancelled, Never Produced` → `注释：已取消，从未生产`
  - `TOW Retired` → `陶式导弹已退役`

## 22. 复合命名的优先级
**综合原则**

当一个条目同时包含多种元素时，按以下优先级处理：

1. **保留**：型号编号、制造商型号、技术规格数字
2. **可选翻译**：北约代号（如有固定译名）
3. **必须翻译**：功能描述、武器类型、状态标识
4. **结构保持**：方括号、圆括号、斜杠的原有结构

示例：
```
原文: 155mm/39 M109A4+ Self-Propelled Howitzer x 6
     ↓
译文: 155mm/39 M109A4+ 自行榴弹炮 x 6门
```

## 翻译示例

### 示例1：复杂飞机命名
```
原文: Su-30MK2 Flanker G
译文: Su-30MK2 Flanker G（侧卫G）
```

### 示例2：带技术规格的武器
```
原文: 127mm/64 Vulcano GLR IR [Guided Long Range]
     Comments: 2015, Land and ASuW, No AAW
译文: 127mm/64 Vulcano GLR IR [制导远程]
     注释: 2015年，陆基和反舰，无防空能力
```

### 示例3：舰船完整命名
```
原文: CVN 65 Enterprise
     Comments: -
译文: CVN 65 企业号
     注释: -
```

### 示例4：复合装备描述
```
原文: Arty Bty (155mm/39 M109A4+ Self-Propelled Howitzer x 6)
     Comments: 2x pr Bty
译文: 炮兵连 (155mm/39 M109A4+ 自行榴弹炮 x 6门)
     注释: 每连2座
```

### 示例5：导弹系统
```
原文: AGM-65D Maverick IR
     Comments: -
译文: AGM-65D 小牛导弹 IR（红外型）
     注释: -
```
